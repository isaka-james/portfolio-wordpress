== Description ==

Portfolio Canvas is a great personal portfolio WordPress theme. You can create a powerful Portfolio website with the Portfolio Canvas theme. There is no doubt that Portfolio Canvas will make your Website look more impressive and attractive. If you are a freelancer, designers, developers, photographers artists and all other professionals and want to build your Portfolio website then Portfolio Canvas is a perfect theme for you. Portfolio Canvas is a very easy-to-use, easily customizable WordPress theme that helps you to build your website in a few minutes. easily customizable WordPress theme that helps you to build your website in a few minutes.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
